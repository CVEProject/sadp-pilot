This folder contains a set of "test set" CVE Records that have been enriched with Supplier Authorized Data Publisher data.    
This data does NOT contain published SADP data (i.e, you will not find them in the official CVE List.)  This data was used as part of Phase I
of the CVE Program SADP Pilot.    The SADP Pilot is now in Phase II.   SADP enriched data is now being published in official CVE List (found [here](https://github.com/CVEProject/cvelistV5) ).
For easy pilot SADP data review, an additional folder has been created [here](https://github.com/CVEProject/sadp-pilot/tree/main/Published%20SADP%20Records) 
which contains a copy of published records that have SADP content.
